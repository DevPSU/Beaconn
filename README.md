# Beaconn
Beaconn is a solution for peer-to-peer groups role call and engagement for academia.

## Installation
Use npm in order to install all required dependencies. 
```bash
npm install
```

## Design Stack
Primary: #34465d

Secondary: #2fcc76

Alternative: rgba(255,255,255,0.2)
